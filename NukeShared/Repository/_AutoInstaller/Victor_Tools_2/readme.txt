V!CTOR TOOLS 2.0v1
==================

To install V!ctor Tools 2.0v1 copy the contents of this folder (Install Victor Tools 2) into the .nuke folder in your system (typically the .nuke folder is located hidden inside your user folder).

If you already have a ‘menu.py’ and/or ‘init.py’ do not overwrite it with the ones contained in the Install folder of the V!ctor Tools, instead open your existing ‘menu.py’ and/or ‘init.py’ and copy and paste there, at the very bottom of the text file the contents from the ‘menu.py’ and/or ‘init.py’ in this Install folder. ‘menu.py’ and ‘init.py’ are editable with any text editor application.

You can delete this ‘readme.txt’ file from the .nuke folder once you copied all the contents.
