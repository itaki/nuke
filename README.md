Installation
-
1. Unzip this repo to somewhere accessible.
2. Open or create a new text document.
3. Add these lines.
```
import os
nuke.pluginAddPath( '{/path/to/this/folder' )
```
4. Save to your `.nuke` folder:
    * Windows: `C:\Users\{user}\.nuke`
    * Linux & Mac: `/home/{user}/.nuke`
